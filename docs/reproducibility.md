# Reproducibility Notes

This repository is a code-only public snapshot of the 1D quantum Ising ED
project. It contains source code, drivers, plotting scripts, documentation,
and empty directory skeletons, but not generated numerical data or final
figures.

## Public Snapshot

Generated data under `data/` and generated figures/media under `plots/` are
not versioned. The repository keeps only `README.md` files and `.gitkeep`
markers in those trees so the expected output layout is visible after clone.

A clean public snapshot should not contain generated tables, arrays, PDFs,
images, videos, logs, work directories, build objects, or retired public-scope
paths.

## Directory Skeletons

The public skeleton includes the output locations used by supported pipelines:

- `data/h_null/observables/`
- `data/h_null/chiz_fd/`
- `data/h_null/fss/`
- `data/quench/`
- `data/h_field/hfield_raw/`
- `data/h_field/hfield_processed/`
- `data/h_field/gh_surface/grids/`
- `data/h_field/gh_surface/raw/`
- `data/h_field/gh_surface/processed/`
- `data/spectral/raw/`
- `data/spectral/processed/`
- `plots/h_null/`
- `plots/quench/`
- `plots/hfield/`
- `plots/spectral/`

Empty directories are kept with `.gitkeep`.

## Supported Public Pipelines

The public code supports:

- h=0 full ED and Lanczos sweeps
- finite-difference longitudinal susceptibility at h=0
- h=0 finite-size scaling and observable plots
- longitudinal h-field CQT/FOQT grids from `run_hfield_grid.py`
- gh-surface data and plots
- ED quench CQT/FOQT/Loschmidt campaigns
- ED-only spectral data generation with `ising_spectral`
- spectral plotting of already generated raw data with `scripts/spectral/plot_spectral.py`

The public build targets are:

```bash
make all
make static quench lanczos spectral hfield ghsurface chizfd
```

## Local Extensions

Some local h-field extension generators and their data products are
intentionally not part of the public repository. Public h-field plots use the
data generated by `scripts/h_field/compute/run_hfield_grid.py` plus processed
tables written by `scripts/h_field/plot_hfield.py` itself.

Local extension data are disabled by default. Advanced local runs may pass
`--use-local-extensions` to let `plot_hfield.py` look for local-only
`mz_ext`/`gap_ext` data if present. This flag is not required to reproduce the
public pipeline, and the public repository does not ship extension generators,
extension binaries, extension source trees, or extension data folders.
